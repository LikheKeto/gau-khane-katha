# Ukhan Tukka - Nepali Riddle Game

A Wordle-style game for Nepali riddles (Ukhan Tukka) with two play modes: free play and daily riddle.

## Project Structure

```
ukhan-tukka/
├── packages/
│   ├── data-processing/     # OCR and data cleaning scripts
│   ├── backend/             # Backend API (TBD)
│   └── frontend/            # Svelte frontend (TBD)
└── README.md
```

## Packages

### Data Processing
Tools for extracting Ukhan Tukka riddles from PDF files, cleaning and structuring the data.

### Backend (Coming Soon)
API for serving riddles and managing game state.

### Frontend (Coming Soon)
Svelte-based UI for the riddle game with free play and daily riddle modes.

## Game Features

- **Free Play Mode**: Solve as many riddles as you want
- **Daily Riddle Mode**: One riddle per day
- **Self-verification**: Users verify their own answers (as Ukhan Tukka answers can vary)
